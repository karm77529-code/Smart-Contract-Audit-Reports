# Smart Contract Security & Audit Portfolio

Welcome to my independent security audit repository. This portfolio showcases my technical expertise in auditing smart contracts, analyzing tokenomics, and detecting vulnerabilities within emerging blockchain ecosystems, with a primary focus on **Solana** and **The Open Network (TON)**.

## 🔍 Auditing Methodology & Tools
Every audit report in this repository is generated through a combination of manual code analysis and advanced technical security tools to evaluate contract risks, liquidity locks, and centralization vectors. My analysis workflow utilizes:
*   **Axiom & Arkham Intelligence:** For tracking on-chain flow, whale activity, and developer wallet links.
*   **RugCheck & Token Sniffer:** For fast, high-fidelity metadata validation and honeypot detection.
*   **Manual Auditing:** Verifying mint functions, ownership status, and liquidity structures.
